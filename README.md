Hyperlocal Emergency Help App

A real-time emergency alert app where users can request help and nearby users get instant notifications.This app can help people in some problem or emergency.

## Tech Stack

- **Frontend:** React + Vite + Tailwind CSS + Socket.IO
- **Backend:** Node.js + Express + MongoDB + Socket.IO
- **Cache:** Redis
- **DevOps:** Docker + GitHub Actions + AWS EC2
