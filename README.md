Hyperlocal Emergency Help App

![CI](https://github.com/123rahulkr/emergency-app/actions/workflows/ci.yml/badge.svg)
![Deploy](https://github.com/123rahulkr/emergency-app/actions/workflows/deploy.yml/badge.svg

A real-time emergency alert app where users can request help and nearby users get instant notifications.This app can help people in some problem or emergency.hurry

## Tech Stack

- **Frontend:** React + Vite + Tailwind CSS + Socket.IO
- **Backend:** Node.js + Express + MongoDB + Socket.IO
- **Cache:** Redis
- **DevOps:** Docker + GitHub Actions + AWS EC2
